# Code of conduct

This project expects straightforward, respectful collaboration.

## Our standard

When you participate here, you agree to:

- discuss the work, not the person
- explain disagreements with evidence
- leave room for other contributors to learn
- avoid harassment, threats, and personal attacks
- respect boundaries around security reports and private disclosures

## Unacceptable behavior

This includes:

- harassment or intimidation
- hate speech
- doxxing
- sustained bad-faith disruption
- sexual language or attention in project spaces
- publishing private security details before a fix exists

## Maintainer responsibility

Maintainers may edit, remove, or reject comments, issues, pull requests, or users that damage the project space.

## Reporting

If someone crosses the line, contact the maintainers through GitHub. The current maintainer file is [MAINTAINERS.md](MAINTAINERS.md).

Keep reports specific. Include links, screenshots, or timestamps when you have them.
